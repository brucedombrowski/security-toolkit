# LaTeX Templates

PDF generation templates for compliance documentation.

## Templates

| Template | Generated By | Purpose |
|----------|--------------|---------|
| `scan_attestation.tex` | `generate-scan-attestation.sh` | Scan results attestation with checksums |
| `security_compliance_statement.tex` | `generate-compliance.sh` | Formal security compliance statement |

## Assets

| File | Purpose |
|------|---------|
| `logo.png` | Organization logo for PDF headers |

## Requirements

PDF generation requires pdflatex (TeX Live or MiKTeX):

```bash
# macOS
brew install basictex

# Ubuntu/Debian
sudo apt install texlive-latex-base texlive-latex-recommended

# CentOS/RHEL
sudo yum install texlive-latex
```

## Customization

To customize templates for your organization:

1. Replace `logo.png` with your organization's logo
2. Edit `.tex` files to modify headers, footers, or formatting
3. Update contact information and organization name

Templates use standard LaTeX commands. See the [LaTeX documentation](https://www.latex-project.org/help/documentation/) for syntax reference.

## Generated PDFs

PDFs are generated in the target project's `.scans/` directory:

- `scan-attestation-YYYY-MM-DDTHHMMSSZ.pdf` - Timestamped scan attestation
- `security-compliance-statement.pdf` - Compliance statement

These PDFs include:
- Scan timestamps and checksums
- NIST control mapping
- Toolkit version and commit hash
- Digital attestation suitable for compliance submittals
